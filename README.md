# Claude Code Memory Tools

Human-like memory skills for Claude Code - store and recall coding patterns and project-specific knowledge.

## Getting Started

1. **Copy this directory to your working project**
2. **Open Claude Code and tell it**: "Execute install.md in this directory"
3. **Restart Claude Code** (exit and re-enter)
4. **Create project-level memory** Run command /create-project-memory-skills to create it (the global level "Coder's Memory" is automatically create already)

Done! Use `--learn` or `--recall` after completing tasks  to build your memory. OR, let it build-up automatically by just using claude-code (It's a claude Skill: memory !)
